## The Core Distinction: Integrator vs. Extender

Zapier is an “Integrator.” It connects the existing, pre-defined features of different apps. It says, “When this button is pressed in App A, do this thing in App B.”

A human strategist using tools like Google Apps Script is an “Extender.” It lets you create brand-new, custom features that don’t exist in the apps in the first place.
